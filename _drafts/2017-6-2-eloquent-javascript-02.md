# 第二章 程序结构

## 2.1 表达式和语句

**表达式**：产生值的操作的代码片段。

**语句**：表达式 + 分号 = 一条语句，在掌握何时省略语句结尾的分号以前，应遵循以分号为结尾的规则。

```javascript
// 这样的表达式不对外界产生任何影响
// 一条语句只有对外部产生影响才有意义
1；
!false;
```

## 2.2 变量

可以使用任意不是**关键字**的单词作为变量名，变量名可以包含数字（不能以数字开头）、字母、下划线，除了$和_字符以外，不能包含其他任何标点符号。

可以将变量想象为许多的触须，而不是一个箱子。

1. 变量并不包含值，而是会引用这些值

1. 两个不同的变量可以引用同一个值

1. 程序只能访问仍被引用的值

1. 当需要保存一些状态的时候，可以生出一条新触须或者伸出一条触须，将其指向某个值
